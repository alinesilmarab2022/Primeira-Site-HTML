iFood disponibiliza 1,7 mil vagas em cursos grátis de tecnologia

O iFood recebe, até está sexta-feira (15), as inscrições para cursos gratuitos de tecnologia e empreendedorismo
oferecidos em parceria com a Oracle. Há 1,7 mil vagas disponíveis, que podem ser preenchidas por pessoas de
todas as partes do Brasil, uma vez que as aulas acontecem via internet.

A oferta contempla cursos online grátis do programa Oracle Next Education (ONE) , ministrados por meio do
Potência Tech, plataforma de educação que incentiva a formação de profissionais de tecnologia.

Clique aqui para ver mais...
O projeto é dividido em quatro trilhas, uma delas com o curso de Lógica de Programação , que abre as portas
para o mercado de desenvolvimento de software.

Também há os cursos de Front-End, que aborda HTML, CSS e JavaScript, e de Java, no qual os alunos vão conhecer
uma das linguagens de programação mais populares do mundo, usada no desenvolvimento de apps e jogos.
O programa tem ainda o curso de Empreendedorismo, com as habilidades essenciais para uma carreira de sucesso.

Quem pode se inscrever?

Além de ser entregador(a) cadastrado(a) no iFood há pelo menos três meses, a pessoa interessada precisa atender a alguns requisitos:

=> Ter 18 anos ou mais;
=> Ter ensino médio completo;
=> Não ter recebido bolsas de estudo da plataforma em tecnologia anteriormente.

Processo seletivo do iFood

Clique aqui para ver mais...


O processo seletivo, assim como o curso, é online. Sendo assim, a seleção acontece em três etapas:

=> Inscrições: até 17 de julho;
=> Minicurso e exercício prático: até 21 de julho;
=> Entrevista: até 25 de julho.




Em geral, o curso tem duração de 30 semanas, com aulas práticas e teóricas. Entre os principais conteúdos, os aprovados desenvolverão conhecimento em linguagens de programação e banco de dados JavaScript com Node.js.
Além disso, o curso oferecerá mentorias para o desenvolvimento de soft-skills, que são habilidades comportamentais.
Entre elas, treinamento para entrevistas de emprego, comunicação verbal e escrita, negociação e conhecimentos básicos sobre o LinkedIn.
Segundo o iFood, a previsão de início das aulas é o dia 8 de agosto.
